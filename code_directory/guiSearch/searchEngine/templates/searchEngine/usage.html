Usage

     Welcome to the GUI Search Engine! GSE is a new tool for Android Developers to find new ideas for GUIs. All you have to do is upload XML from your repository and click search. There are four different algorithms you can choose from. The default is by components, which does one thing, then there are 3 others.

     Resuls are displayed by similarity to the original file(s). The name of the app will be displayed, followed by the closest XML files to the ones inputted by the user, and the number of components in that file. Also displayed are the number of components in the XML files the user inputted, for comparison. Click here to go to search screen.
